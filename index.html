What is Electronify?

Electronify is an application which purpose is to create Electron projects easily,
serving as an interface between the user and the commands needed to create an Electron project.

· Create a project

In order to create your Electron project, you'll need to fill in these fields. Here is the explanation of each one of them:

  • > Project name ➖ States the project's name.

  • > Project description ➖ States the project's description. It's supposed to be brief.

  • > License ➖ If the project will be made under a specific license, it's stated into this field.

  • > Version ➖ Current version of the project when creating it.

  • > Author ➖ States the project's author name. It could be a single person or a company.

  • > Test command ➖ Sets up a specific command used to test the project.

  • > GIT Repository ➖ User and repository on GitHub for the project.

  • > Path of the project ➖ Specific local path for the project to be stored in.


Some of these fields have default values already set, in case you don't fill them up.

With these fields filled, the only step left is to click on the Create button,
and your Electron project will be on its way to be created.
